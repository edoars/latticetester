Detailed Description

template<typename RedDbl>
class LatticeTester::NormaRogers< RedDbl >

This class implements the Rogers bounds on the density of sphere packing.

The length of vectors is computed using the \({\mathcal{L}}_2\) norm. The bounding lengths, for a lattice containing \(n\) points per unit volume in dimension \(t\), are given by \(\ell_t^* = \gamma_t^{1/2} n^{-1/t}\), where the \(\gamma_t\) are the Rogers lattice constants.
